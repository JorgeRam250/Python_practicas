# Distrito Chilaquil — Logo Kit (Opción 1: Mascota Totopo)

Este kit contiene 3 SVG listos para UI:
- `icon_color.svg` (color)
- `icon_mono_black.svg` (monocromo negro)
- `icon_mono_white.svg` (monocromo blanco)

## Paleta recomendada (tokens)
- Maíz (base): #F6C445
- Salsa roja (acento): #E53935
- Carbón (texto/ojos): #1F1F1F
- Crema (fondo): #FFF6E8
- Verde salsa (acento opcional): #2E7D32

## Uso recomendado en UI
- Navbar (24–32px de alto): usa **wordmark + ícono** (si agregas texto en Figma).
- Sidebar / botones / app icon: usa **solo ícono**.
- Favicon: simplifica (solo ojos + gota, sin sonrisa) para 16px.

## Reglas de consistencia
- Mantener el ícono como forma sólida (evitar texturas).
- No añadir más de 1 sombra ligera (si se usa, que sea sutil y consistente).
- Clear space: mínimo 1/4 de la altura del ícono alrededor.
- Para tamaños pequeños (≤24px): quitar detalles (highlight de gota y/o sonrisa).

## Tipografías sugeridas
- Marca (títulos): Baloo 2 o Fredoka
- UI (texto): Inter
